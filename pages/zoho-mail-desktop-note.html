Zoho Mail Desktop client

by Arne Govaerts

On Zoho Mail, IMAP is only accessible for premium users, which I'm not (cheapish student that I'm :P ), so I'm/was stuck with their webmail client. Still a fair deal, but why not create a desktop client myself?
These days, this takes under an hour or so, or even faster for simply showing a website. Enter Electron. Being a extremely stripped out version of Chromium, it most basic future is browsing a single website.
The icon is still a bit of a problem. Normally you should be able to change the icon of an executable to something of your own with a tool like rcedit, but it refused to work, too bad. I made desktop link (in Windows 10) and changed that icon. Not as good, but same outside look.

The result can be found here.
